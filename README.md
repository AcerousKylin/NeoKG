# NeoKG

NeoKG is a Neo4j-based Django REST Framework Application of graph visualization using neo4j official lib, neomodel, echarts and its subs, bootstrap and so on.

This is source code repository of AcerousKylin's Capstone Project with title "Construction and Application of Knowledge Graph facing Technology Service Industry". You can feel free to have any clip of these codes with reasonable tunes, modifications or reductions, because it's not for commercial purpose but academic purpose and meant to be open source while I have the same dilemma in constructing it.
